# 类型 标题 发生频率 影响 描述
P1 :woman_technologist: Web应用漏洞 高 非常高 在任何保护或运行着敏感用户数据的系统中，漏洞都是一个关键难题。未能恰当的设计和实现应用，未能检测到应用存在的问题或未能快速对检测到的问题进行修复（或打补丁）都可能导致隐私被泄露。该风险还包括OWASP Top 10列出的web应用漏洞及其他风险。
P2 :man_office_worker::woman_technologist: 运营方数据泄露 高 非常高 未能防止包含用户及相关信息的数据泄露给未经授权方，导致数据保密性的丧失。由于故意破坏或无意的错误（如不到位的访问管理控制，不安全的存储，不必要的数据复制或缺乏安全意识）而引入。
P3 :man_office_worker::woman_technologist: 数据泄漏响应不足 高 非常高 不向受影响人（数据主体）通知可能因有意或无意的安全事件产生的数据泄漏；失败的补救措施；没有试图限制泄漏。
P4 :man_office_worker: 全部同意 非常高 高 将条款混在一起或不恰当地让用户同意业务提供方后续的一切处理。同意应是“逐条”确认的，而不是汇在一起（如将使用网站和用于广告分析的条款混在一起）。
P5 :man_office_worker: 不透明的政策、条款和条件 非常高 高 没有提供足够的信息来描述数据是如何被处理的，例如数据的收集、储存、处理。而且非律师身份用户难以理解这些信息。
P6 :man_office_worker::woman_technologist: 个人数据删除不完整 高 高 在特定目的或请求完成后，未能及时有效删除个人数据。
P7 :man_office_worker::woman_technologist: 数据质量不足 中 非常高 使用过期、不正确或虚假的用户资料。未能更新或更正数据。
P8 :woman_technologist: 会话过期机制缺少或不足 中 非常高 未能有效终止会话。可能会导致未经用户同意或在用户不知情的情况下收集额外的用户数据。
P9 :man_office_worker::woman_technologist: 用户无法访问和修改数据 高 高 用户不能访问、更改或删除与自己相关的数据。
P10 :man_office_worker: 未经用户同意收集数据 高 高 收集与系统无关的描述性、人口统计或任何其他用户数据。同样适用于未经用户同意的数据。
