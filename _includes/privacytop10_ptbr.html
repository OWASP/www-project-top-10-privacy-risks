# Tipo Título Frequência Impacto Descrição
P1 :woman_technologist: Vulnerabilidades em aplicações web Alto Muito Alto A vulnerabilidade é um problema importante em qualquer sistema que protege ou opera com dados confidenciais do usuário. A falha em projetar e implementar adequadamente um aplicativo, detectar um problema ou aplicar imediatamente uma correção (patch) provavelmente resultará em uma violação de privacidade. Esse risco também abrange a lista do OWASP Top 10 e os seus riscos resultantes.
P2 :man_office_worker::woman_technologist: Vazamento de dados pelo lado do operador Alto Muito Alto Falha em evitar o vazamento de qualquer informação que contenha ou esteja relacionada aos dados do usuário, ou os próprios dados, para qualquer parte não autorizada, resultando na perda da confidencialidade dos dados. Introduzido devido a violação maliciosa intencional ou erro não intencional, por exemplo causados por controles de gerenciamento de acesso insuficientes, armazenamento inseguro, duplicação de dados ou falta de conscientização.
P3 :man_office_worker::woman_technologist: Resposta insuficiente devido ao vazamento de dados Alto Muito Alto Não informar as pessoas afetadas (titulares dos dados) sobre uma possível violação ou vazamento de dados, resultante de eventos intencionais ou não; falha em remediar a situação corrigindo a causa; não tentando limitar os vazamentos.
P4 :man_office_worker: Consentimento em tudo Muito Alto Alto Agregação ou uso impróprio de consentimento para processamento legítimo. O consentimento é "sobre tudo" e não é coletado separadamente para cada finalidade (por exemplo, uso do site e criação de perfil para publicidade).
P5 :man_office_worker: Políticas, termos e condições não transparentes Muito Alto Alto Não fornecendo informações suficientes para descrever como os dados são processados, como sua coleta, armazenamento e processamento. Falha em tornar essas informações facilmente acessíveis e compreensíveis para não advogados.
P6 :man_office_worker::woman_technologist: Exclusão insuficiente de dados pessoais Alto Alto Falha na exclusão efetiva e/ou oportuna de dados pessoais após o término da finalidade especificada ou mediante solicitação.
P7 :man_office_worker::woman_technologist: Qualidade dos dados insuficiente Médio Muito Alto O uso de dados do usuário desatualizados, incorretos ou falsos. Falha ao atualizar ou corrigir os dados.
P8 :woman_technologist: Expiração de sessão ausente ou insuficiente Médio Muito Alto Falha em impor o encerramento da sessão de forma eficaz. Pode resultar na coleta de dados adicionais do usuário sem o consentimento ou conhecimento do usuário.
P9 :man_office_worker::woman_technologist: Incapacidade dos usuários de acessar e modificar seus dados Alto Alto Os usuários não têm a capacidade de acessar, alterar ou excluir dados relacionados a eles.
P10 :man_office_worker: A coleta de dados não requeridos para o propósito consentido pelo usuário Alto Alto Coletar dados descritivos, demográficos ou quaisquer outros dados relacionados ao usuário que não sejam necessários para os propósitos do sistema. Aplica-se também a dados para os quais o usuário não deu consentimento.
